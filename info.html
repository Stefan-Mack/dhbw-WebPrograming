Infos zur Webseite

API

Für die Webseite wurde eine Community API von ByMykel verwendet. Die zwei API-Calls die hier verwendet wurden sind:

  • https://bymykel.github.io/CSGO-API/api/de/skins.json für die Startseite
  • https://cs2-api.vercel.app/api/items?id=${urlId} für die individuellen Items

Probleme

Die API brachte auch verschiedene Herausforderungen mit sich. So gab es zwei verschiedene Versionen mit unterschiedlichen Datenstrukturen. Da sich die erste API mehr auf Skins fokussiert und die 2. auf alle Ingame-Items, habe ich mich bei der Startseite auf diese fokussiert. Die IDs sind jeweils gleich auf beiden APIs, deshalb wurde für die Details die neuere verwendet. Eine Suchfunktion brachte die neue API auch mit, aber war wegen der Datenstruktur nicht

Warum diese API?

Da ich am gamen interessiert bin, sollte es in diese Richtung gehen. Die suche nach einer entsprechenden API so zwar auf den ersten Blick ziemlich einfach aus. Es kam viel aus der Community, aber diese waren dann doch sehr beschränkt in ihrere Nutzung. Für viele musste man etwas da lassen, andere waren nicht nutzbar, da diese nicht CORS zugelassen haben. Da änderte sich dann die API im Laufe der Entwicklung mehrmals.
