Steam

Steam ist eine Spieleplattform von Valve, auf der es von Shooter über Rollenspiele bis Simulationen alles an Games gibt. Die benutzte API kann Daten zu allen Skins im Spiel liefern

Counter Strike

Counter Strike ist ein Shooter auf Steam. Das Spiel beinhaltet Waffenskins, die auf dem Steam Market in großer Menge gehandelt werden.

Ziel

Die Webseite soll diese Skins per API abrufen, um eine Oberfläche zu liefern, auf der man sich alle Skins anschauen kann
